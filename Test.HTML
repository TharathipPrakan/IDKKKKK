HTML และ CSS ที่ใช้บ่อย

พื้นฐาน

  • <!DOCTYPE html>: ประกาศเอกสาร HTML5
  • <html>: เริ่มต้นและสิ้นสุดเอกสาร HTML
  • <head>: ส่วนหัวของเอกสาร
  • <title>: ชื่อเรื่องของเอกสาร
  • <meta>: เมตาดาต้า เช่น การเข้ารหัสอักขระ, ข้อมูล viewport
  • <link>: ลิงค์ไปยังไฟล์ภายนอก เช่น CSS
  • <style>: สไตล์ในตัว
  • <script>: สคริปต์ภายใน หรือเชื่อมโยงกับไฟล์สคริปต์ภายนอก
  • <body>: เนื้อหาหลักของเอกสาร

ข้อความและรูปแบบ

  • <h1> - <h6>: หัวเรื่องระดับ 1 ถึง 6
  • <p>: ย่อหน้า
  • <br>: ขึ้นบรรทัดใหม่
  • <hr>: เส้นแบ่งแนวนอน
  • <strong>: ข้อความเข้ม
  • <em>: ข้อความเอียง
  • <b>: ข้อความหนา
  • <i>: ข้อความเอียง
  • <u>: ขีดเส้นใต้ข้อความ
  • <mark>: ข้อความเน้น
  • <small>: ข้อความขนาดเล็ก
  • <del>: ข้อความที่ถูกลบ
  • <ins>: ข้อความที่ถูกแทรก

การจัดโครงสร้าง

  • <div>: กล่องคอนเทนต์ (block)
  • <span>: กล่องคอนเทนต์ (inline)
  • <header>: ส่วนหัว
  • <footer>: ส่วนท้าย
  • <nav>: การนำทาง
  • <section>: ส่วนหรือหมวด
  • <article>: บทความ
  • <aside>: เนื้อหานอกข้าง
  • <main>: เนื้อหาหลัก
  • <figure>: รูปภาพหรือแผนภูมิ
  • <figcaption>: คำบรรยายรูปภาพหรือแผนภูมิ

ลิงค์และรูปภาพ

  • <a>: ลิงค์
  • <img>: รูปภาพ
  • <map>: แผนที่รูปภาพ
  • <area>: พื้นที่ในแผนที่รูปภาพ
  • <picture>: รูปภาพที่ตอบสนอง
  • <source>: แหล่งที่มาของรูปภาพหรือมัลติมีเดีย

ตาราง

  • <table>: ตาราง
  • <tr>: แถวในตาราง
  • <th>: หัวข้อในตาราง
  • <td>: เซลล์ข้อมูลในตาราง
  • <thead>: ส่วนหัวของตาราง
  • <tbody>: ส่วนเนื้อหาของตาราง
  • <tfoot>: ส่วนท้ายของตาราง
  • <caption>: คำอธิบายของตาราง
  • <col>: กำหนดสไตล์สำหรับคอลัมน์
  • <colgroup>: กลุ่มคอลัมน์ในตาราง

ฟอร์ม

  • <form>: แบบฟอร์ม
  • <input>: ช่องกรอกข้อมูล
  • <textarea>: กล่องข้อความ
  • <button>: ปุ่ม
  • <select>: รายการตัวเลือก
  • <option>: ตัวเลือกในรายการ
  • <label>: ป้ายชื่อสำหรับฟอร์ม
  • <fieldset>: กลุ่มฟอร์ม
  • <legend>: คำอธิบายกลุ่มฟอร์ม
  • <datalist>: รายการข้อมูล
  • <output>: ผลลัพธ์ของฟอร์ม

มัลติมีเดีย

  • <audio>: เสียง
  • <video>: วิดีโอ
  • <source>: แหล่งที่มาสำหรับ <audio> หรือ <video>
  • <track>: แทร็กสำหรับ <audio> หรือ <video>
  • <embed>: ฝังมัลติมีเดีย
  • <object>: ฝังวัตถุ
  • <param>: พารามิเตอร์สำหรับ <object>
  • <iframe>: แทรกเฟรม

การจัดกลุ่มเนื้อหา

  • <ul>: รายการไม่เรียงลำดับ
  • <ol>: รายการเรียงลำดับ
  • <li>: รายการในรายการ
  • <dl>: รายการคำอธิบาย
  • <dt>: คำในรายการคำอธิบาย
  • <dd>: คำอธิบายในรายการคำอธิบาย
  • <figure>: รูปภาพหรือมัลติมีเดียที่มีคำอธิบาย
  • <figcaption>: คำอธิบายสำหรับ <figure>
  • <main>: เนื้อหาหลักของเอกสาร
  • <div>: กล่องคอนเทนต์ทั่วไป

อื่นๆ

  • <details>: รายละเอียดเพิ่มเติมที่สามารถยุบและขยายได้
  • <summary>: สรุปสำหรับ <details>
  • <menu>: เมนู
  • <menuitem>: รายการในเมนู
  • <dialog>: กล่องสนทนา
  • <canvas>: พื้นที่สำหรับการวาดภาพด้วย JavaScript
  • <svg>: รูปภาพเวกเตอร์
  • <math>: สัญลักษณ์ทางคณิตศาสตร์
